Installation¶

How to install?¶

Installation¶

conda create -n poem_libs python=3.10
conda activate poem_libs
pip install poem-ravenframework

Clone¶

git clone git@github.com:idaholab/POEM.git

Test¶

cd POEM/tests
poem -i lhs_sampling.xml
